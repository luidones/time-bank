# Banco do Tempo

### Descrição
Aplicação que controla as atividades realizadas entre usuários de um banco do tempo onde a moeda são as horas gastas na troca dos serviços.


### Entidades

- Usuário
    - Talentos
        - Reviews
        - Agenda
            - Disponibilidade
            - Reserva
- Banco
- Conta
- Transações

### Tec

- GUI VueJs
- API Restify (Nodejs)
- DB Postgres
